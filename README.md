## Getting Started

First, install the project:

```bash
git clone git@github.com:arnolanglade/mikado-method.git

volta setup
pnpm setup
pnpm install
```

Then, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
